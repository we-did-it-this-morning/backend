Every request except for user management require the token appended to the url in the GET parameters:
GET /country?token=abcdef

POST type must be application/json

If an error occurs the following will be returned:
{
  success: false;
  message: string; // e.g. already here
}
Otherwise
{
  success: true;
  ...
}


--- user management ---
POST /log-in
{
  username: string;
  password: string;
}

Returns
{
  token: string;
}

POST /register 
{
  username: string;
  password: string;
}


--- countries ---


POST /update-country
{
  id?: number;
  name: string;
  malariaTypes: number[];
}

GET /countries

Returns
Array<{
  id: string;
  name: string;
  malariaTypes: number[];
}>
