Malaria API


Basic Info

 
POST type must be application/json

All routes that needs authentication, require the token appended to the url in the GET parameters:
/update-country?token=abcdef    
(Put the other expected data in body)

If an error occurs the following will be returned:
{
    success: false,
    message: string
}
Otherwise
{
    success: true,
    data: NULL || { Object } || [ Array ]
}
